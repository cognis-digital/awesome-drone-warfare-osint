# Theaters of operation

The same forensic schema applies to every theater the dataset covers.

| Theater | Status | Primary actors |
|---|---|---|
| [🇺🇦 Russia–Ukraine](ukraine.md) | **Primary corpus** | RU/IR drones vs. UA defenses + UA strike drones |
| [Middle East](middle-east.md) | Active | Israel/Hamas/Hezbollah/Iran/Houthi |
| [Red Sea / Yemen](red-sea.md) | Active | Houthi vs. coalition shipping |
| [Sudan civil war](sudan.md) | Active | RSF vs. SAF (Chinese, Turkish, Iranian drones) |
| [Myanmar](myanmar.md) | Active | PDF / EAOs vs. junta (3D-printed UAS) |
| [Indo-Pacific](indo-pacific.md) | Posture / exercises | PLA / Taiwan Strait |
| [Nagorno-Karabakh](nagorno-karabakh.md) | Reference (2020) | Azerbaijan TB2 / Israeli, Armenia Krunk |
| [Libya](libya.md) | Active (2019–) | LNA (Wing Loong II) vs. GNA (Bayraktar TB2) |

Every theater inherits the same `incidents.csv` schema. The `incidents.csv`
table itself is intentionally not redistributed; instead, the schema is
documented so users can join their own ACLED extract.
