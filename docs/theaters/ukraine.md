# 🇺🇦 Russia–Ukraine theater

The primary corpus of this dataset. Notable developments:

- **Operation Spider's Web** (June 2025) — Ukrainian deep-strike operation
  that destroyed strategic-aviation airframes deep inside Russia using FPV
  drones built on open-source ArduPilot autopilot software.
  [CSIS analysis](https://www.csis.org/analysis/how-ukraines-spider-web-operation-redefines-asymmetric-warfare).
- **Shahed scaling** — from dozens per month (2022) to thousands per month
  (2025+), now produced inside Russia as the Geran-2/3/4/5.
- **Mesh networking** — Russian focus on mesh-enabled C2 (per
  [Ukrainian Arms Monitor](https://ukrainesarmsmonitor.substack.com/p/drone-warfare-in-ukraine-baba-yaga)).
- **Counter-UAS arms race** — Ukrainian "Sting" interceptor FPVs vs.
  Russian Geran swarms.

See `data/components.csv` for the underlying corpus (4,790+ rows).
