## 🛠 Open-source flight stacks

These are the autopilots that power most non-DJI drones in the war
(including Ukraine's [Operation Spider's Web](https://www.csis.org/analysis/how-ukraines-spider-web-operation-redefines-asymmetric-warfare)):

- [ArduPilot](https://github.com/ArduPilot/ardupilot) — used in Ukrainian long-range strike (canonical)
- [PX4 Autopilot](https://github.com/PX4/PX4-Autopilot) — Auterion-backed
- [Betaflight](https://github.com/betaflight/betaflight) — FPV racing → combat
- [iNav](https://github.com/iNavFlight/inav) — fixed-wing & long-range
- [Cleanflight](https://github.com/cleanflight/cleanflight)
- [AuterionOS](https://auterion.com/product/auterion-os/) — commercial PX4 fork
- [NORDA Dynamics](https://ukrainesarmsmonitor.substack.com/p/towards-greater-drone-autonomy-norda) — UA autonomy stack for Raspberry Pi

This repo does **NOT** host or mirror weaponizable firmware variants —
only links to canonical upstreams. See [`DISCLAIMER.md`](../DISCLAIMER.md).
