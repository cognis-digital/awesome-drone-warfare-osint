# Drone of the Week

Each Monday at 14:00 UTC, the `.github/workflows/dotw.yml` workflow opens
a new issue profiling one platform. The format is fixed so screenshots
are consistent on social channels.

## Template

```markdown
# Drone of the Week #<N> — <Display Name>

**Operator(s):** <flag(s)>
**Role:** <role>
**First observed:** <YYYY-MM-DD>
**Documented components in dataset:** <count>
**Primary source:** <url>

## Hero image
![photo](<url>)

## Top-10 documented components
| # | Marking | Manufacturer | Country | Function |
|---|---|---|---|---|
| 1 | ... | ... | ... | ... |
...

## Strategic significance
3-5 sentences.

## Sources
- ...
- ...
```
